## 인공지능 (Artificial Intelligence, AI) 이란?

**인공지능(AI)**은 인간의 학습, 추론, 문제 해결, 언어 이해, 인식 등 지적 능력을 컴퓨터 시스템으로 구현하는 기술이다.

---

### 핵심 개념

**좁은 의미 (Narrow AI)**
특정 작업에 특화된 AI. 현재 우리가 사용하는 대부분의 AI가 여기에 해당한다.
예) 이미지 인식, 번역, 바둑, 음성 인식

**넓은 의미 (General AI)**
인간처럼 모든 영역에서 스스로 생각하고 판단하는 AI. 아직 실현되지 않은 미래 목표.

---

### 주요 분류

| 분류 | 설명 | 예시 |
|------|------|------|
| **머신러닝(ML)** | 데이터를 학습해 스스로 규칙을 찾음 | 추천 알고리즘 |
| **딥러닝(DL)** | 신경망 구조로 복잡한 패턴 학습 | ChatGPT, 이미지 생성 |
| **자연어처리(NLP)** | 인간 언어를 이해하고 생성 | 번역기, LLM |
| **컴퓨터 비전** | 이미지·영상을 인식하고 분석 | 자율주행, 얼굴 인식 |

---

### AI의 3가지 학습 방법

1. **지도 학습** — 정답이 있는 데이터로 학습 (예: 스팸 필터)
2. **비지도 학습** — 정답 없이 패턴을 스스로 발견 (예: 고객 군집화)
3. **강화 학습** — 보상/패널티를 통해 최적 행동 학습 (예: 게임 AI)
